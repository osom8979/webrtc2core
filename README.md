# WEBRTC2CORE

[![NPM](https://nodei.co/npm/webrtc2core.png)](https://nodei.co/npm/webrtc2core/)

## Introduction

webrtc2core is a front-end WebRTC client.

## Install package

Install n package-manager.
```bash
yarn global add n
# OR
npm install -g n
# OR
brew install n
# OR
curl -L https://git.io/n-install | bash
```

Change node version:
```bash
n lts
# OR
n 12
# OR
n 12.16
```

Install modules:
```bash
yarn
# OR
npm install
```
